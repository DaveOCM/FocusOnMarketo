Email Performance Report

üìö Official Documentation: Email Performance ‚Üí

Key Metrics

Metric Definition
Sent Total emails sent (delivered + bounced)
Delivered Successfully delivered to inbox
Bounced Failed delivery (hard + soft bounces)
Opened Recipient opened email (downloaded images)
Clicked Recipient clicked a link
Unsubscribed Recipient unsubscribed via this email

Email Metric Formulas

Important Formulas:
Open Rate = (Opens √∑ Delivered) √ó 100%
Click Rate = (Clicks √∑ Delivered) √ó 100%
Click-to-Open Rate = (Clicks √∑ Opens) √ó 100%

Note: Use DELIVERED (not Sent) as denominator. Sent includes bounces which can't open/click.

Click-to-Open Rate vs Click Rate:

  • Click-to-Open Rate: Of people who opened, how many clicked? Measures content effectiveness.
  • Click Rate: Of people delivered, how many clicked? Measures overall campaign effectiveness.

Program Performance Report

üìö Official Documentation: Program Performance ‚Üí

Key Feature: Program Performance Report SUPPORTS filtering by Channels and Custom Tags. This is unique to this report!

Key Metrics

Metric Definition
Total Members All people in program
New Names New people acquired by this program
Success People who achieved program success
Total Cost Sum of all period costs
Cost per Success Total Cost √∑ Success

ROI Calculation

ROI = [(Revenue - Cost) √∑ Cost] √ó 100%

Example:
Revenue: $100,000
Cost: $20,000
ROI = [($100,000 - $20,000) √∑ $20,000] √ó 100%
ROI = 400%

Campaign Activity Report

üìö Official Documentation: Campaign Activity Report ‚Üí

Shows smart campaign processing activity and health.

Key Metrics:

  • Batch Run Count: Number of times campaign ran
  • Total People Processed: People who went through flow
  • People in Wait Step: Currently in wait steps
  • Removed from Flow: People removed before completion

Use Cases:

  • Monitor campaign health
  • Identify campaigns with people stuck in wait
  • Find inactive trigger campaigns
  • Troubleshoot processing issues

Report Subscriptions

üìö Official Documentation: Report Subscriptions ‚Üí

Schedule reports to be emailed automatically on a recurring basis.

Options:

  • Daily, Weekly, Monthly, or Quarterly frequency
  • Choose recipients
  • Select report format (Excel, CSV, PDF)

Smart List Subscriptions

üìö Official Documentation: Smart List Subscriptions ‚Üí

Email a list of people who qualify for a Smart List on a recurring schedule.

Difference from Report Subscriptions:

  • Report Subscription: Sends aggregated metrics/summary data
  • Smart List Subscription: Sends list of actual people with their field values
Important: Smart List Subscriptions send a list of PEOPLE (like an Excel export). Report Subscriptions send METRICS about performance.
